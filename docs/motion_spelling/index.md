## 🔹 拼字主角與題目

----------------------------

: 首先，要先決定拼字的主角與題目，請從Scratch挑選一個喜歡的角色，建議挑選的主角單字在3~5個字母之間，以免難度過高。


: ![pick_word](pick_word.png)

<br/>
<br/>

: 在許多遊戲中，向玩家告知遊戲的目的很重要，所以我們先來設計題目的敘述，如「請拼出DOG」，為了加強效果，我們會使用「逐字說出」，如下圖，想一想，在Scratch中要如何做出這個效果呢？


: ![spell_anime](spell_anime.gif)
