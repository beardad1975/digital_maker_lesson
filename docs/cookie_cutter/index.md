### 🔹 認識造型模具

: 各位同學，你知道餅干是怎麼做出來的嗎？

: 麵粉、粉、奶油、雞蛋等材料做出麵糊， 麵糊放入烤箱就可以做出餅干。在過程中，如果有個可愛的造型模具，就可以讓餅干有各種不同的造型。

: 造型模具可以有不同的材料與製作方式，在這個單元中，我們會使用3D列印設計來製作。

### 🔹 草圖、接線與抽殼

----------------------------

: 要畫出模具的輪廓造型，需要能夠畫出各種線條。接下來一起來練習各種草圖的用法。
: 「**橢圓形**」的草圖，可以畫出如卡通臉形或是耳朵的輪廓。利用兩個不同長短的半徑可以畫出橢圓形。
: ![ellipse_sketch_practice1](ellipse_sketch_practice1.jpg)
: ![ellipse_sketch_practice2](ellipse_sketch_practice2.jpg)

<br/>
<br/>

: 「**多邊形**」的草圖可以調整邊數，如正6邊形或正5邊形。
: ![polygon_sketch_practice1](polygon_sketch_practice1.jpg)
: ![polygon_sketch_practice2](polygon_sketch_practice2.jpg)

<br/>
<br/>


: 「**折線**」的草圖可以畫出任意的多邊形，拖曳時也可以拉出圓弧。
: ![polyline_sketch_practice1](polyline_sketch_practice1.jpg)
: ![polyline_sketch_practice2](polyline_sketch_practice2.jpg)

<br/>
<br/>

: 要注意線條必需要**封閉**(圍成一圈)，有封閉才可以擠出成形。
: ![sketch_closed](sketch_closed.jpg)

<br/>
<br/>

: 「**曲線**」的線條可以設計出不同弧度的的線條。
: ![spine_sketch_practice1](spine_sketch_practice1.jpg)
: ![spine_sketch_practice2](spine_sketch_practice2.jpg)
<br/>
<br/>
<br/>

: 如果想要將各種不同的草圖組合或分割，一定要做「**接線**」的動作。接線時，要先點選前一個草圖。
: ![relay_sketch_practice1](relay_sketch_practice1.jpg)
: ![relay_sketch_practice2](relay_sketch_practice2.jpg)
<br/>
<br/>

: 如果有多餘的線條要刪除，可以使用「**擦拭**」功能。
: ![trim_sketch_practice1](trim_sketch_practice1.jpg)

<br/>
<br/>

: 將設計好的立體形狀，做「**抽殼**」動作，可中間抽空。
: ![shell_practice1](shell_practice1.jpg)
: ![shell_practice2](shell_practice2.jpg)

<br/>
<br/>
<br/>

: 最後，利用「**壓平**」功能，把底面去除。
: ![press_practice1](press_practice1.jpg)
: ![press_practice2](press_practice2.jpg)

<br/>
<br/>
