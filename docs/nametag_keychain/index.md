### 打洞練習

----------------------------

: 鑰匙圈裝飾為了固定的功能，需要在上面打出一個洞。

: 打洞的操作與一個哆啦A夢的道具很像。在漫畫人物哆啦A夢的百寶袋中，有一個神奇的道具是「超空間割刀」。

: 它可以在任何地方做出超空間櫥櫃，用來儲存物品，當作保險箱用。又名「儲物箱刀片」。


: ![哆啦A夢](doraemon_locker_cutter.jpg)
